This is a python project for scrapping college results. This python scripts fetches name,roll number and CG of student or a list of students.
Uses BeautifulSoup and regrex for extracting info.
Store the result in a csv file.
HOW TO USE : 
1) install bs4 and requests modules.
2) run : python results.py
3) Write branch shorthand (bec --> for electronics)
4) Enter starting and ending roll numbers.

This scripts also include random delays to avoid overloading the server.
